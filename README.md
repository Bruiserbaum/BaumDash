BaumDash is a Windows Forms application built in C# on .NET, designed to act as a dedicated control dashboard for a secondary touchscreen. It is optimized for a 13-inch 1920×720 touchscreen running Windows 11, typically positioned below a primary monitor to provide quick, glanceable system control without interrupting your main workflow.

The goal of BombDash is simple: put the most useful system controls, integrations, and information panels in one touch-friendly dashboard.

Core Features
System Audio Controls

Quickly manage your system sound without digging through Windows menus.

Microphone mute toggle

Speaker output selection

Master volume control

Touch-friendly sliders and toggles

Instant Replay Capture

Trigger your GPU’s built-in replay capture with a single tap.

Supports integrations with:

AMD ReLive

NVIDIA ShadowPlay

Perfect for capturing highlights while gaming or streaming.

App Control Modules

BombDash allows custom modules that can interact with specific applications.

Examples include:

Launching apps

Controlling app states

Quick access utilities for frequently used tools

This modular design will expand as more integrations are added.

Media Playback Controls

BombDash hooks into Windows’ system media session, allowing control of whatever is currently playing.

Controls include:

Play / Pause

Skip / Previous

Media title and playback display

Works with most apps Windows detects as a media source (Spotify, YouTube, browser players, etc.).

Time & Weather

Quick status widgets provide at-a-glance information:

Current time and date

Weather conditions and temperature

Discord Integration

Stay connected without switching windows.

BombDash can connect to Discord to display:

Servers

Channels

Basic channel navigation

This allows quick monitoring while gaming or working.

AI Chat Integration

Built-in AI chat support allows BombDash to function as a lightweight assistant panel.

Supported backends include:

AnythingLLM (local or remote)

ChatGPT

This enables quick AI queries without leaving your current application.

PC Performance Monitoring

Keep an eye on system health while working or gaming.

Displays metrics such as:

CPU usage

Memory usage

Other system stats (expandable in future versions)

Google Calendar

Events can be displayed directly on the dashboard.

Switches

Designed for Touch

BombDash was designed specifically for touchscreen interaction, with large buttons and simple controls suited to:

13-inch ultrawide touch displays

1920×720 resolution

Desk-mounted secondary displays

This layout works well for streamers, gamers, developers, and power users who want quick control without alt-tabbing.

Startup Behavior

Default behavior:
